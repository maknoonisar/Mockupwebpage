# Interactive Element Tracking Tool

A simple tool that allows users to select and track interactions on a mock webpage. It can identify and mark elements for tracking and simulate events to a mock Google Tag Manager (GTM) endpoint.

## Features

- Create a mock webpage with interactive elements, including buttons, links, video, and a dropdown.
- Enable selection mode for highlighting and tracking interactive elements.
- Simulate sending events to a mock GTM endpoint.
- Log event data in the console, including element type and label.
- Toggle selection mode to select/unselect elements.
